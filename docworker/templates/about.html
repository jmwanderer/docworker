About DocWorker

DocWorker uses GPT AI models to summarize, analyze, and transform docouments.

ChatGPT and other GPT AI language models deliver amazing functionality by taking text as input and generate text as output. These models can chat with humans, write an email, and answer questions. Howver, GPT AI language models are are limited in the amount of text that can be processed in a single operation. If we want to feed the GPT model a large document, the document needs to be broken up into sections and each section processed individually.

This is what DocWorker does. DocWorker accepts PDF, DOCX, and TXT documents, breaks them into sections, feeds individual sections to the GPT AI model, and combines the results back into one document.

For example, a person may want to extract the key concepts from the Supreme Court decision GROFF v. DEJOY which is published as a PDF file. Using DocWorker, this is done in three steps:

  • Upload the groff-dejoy.pdf file
  • Enter a prompt for the GPT operation, "Describe the key concepts"
  • Run the operation and wait for the result

Example Operations

The GPT AI models can perform many types of operations, including:

  • Summarize a document
  • Extract people, places, dates
  • Extract key concepts
  • Rewrite a document for clarity or directed to a target auidence
  • Explain concepts contains in a document
  • Translate a document

Consolidate vs Transform

A typical document processing operation is one that consolidates information from the entire document. For example, 'summarize' or 'explain the main points', starts with a large document and produces result much shorter than the original document.

However some operations transform a document and produce a result of similar size. Example operations are to translate or re-write. These are considered to be transform operations that don't reduce or summarize the original document.

DocWorker allows users to select either Consolidate (the default) or Transform, depending on the desired result.

Document Query

DocWorker processes documents to rewrite or summarize the contents. It is not designed to answer specific questions using documents as a knowledge base. A good option for this type of functionality is ChatPDF
